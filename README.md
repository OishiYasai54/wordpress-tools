# WP-Site-Tool

サーバー内の `/var/www` を自動スキャンし、稼働中のWordPressサイトの情報一覧表示。

## 取得可能な情報

- サイト名 (Blog Name)
- サイトURL
- インストールパスとディレクトリ所有者
- WordPress バージョン
- 管理者ユーザー名とメールアドレス
- 有効化されているテーマ
- 有効化されているプラグイン一覧（カンマ区切り）

##　スクリプトの使い方

### 1. 前提条件

- `wp-cli` がインストールされていること。
- スクリプトを実行するユーザーが `sudo` 権限を持っていること（root推奨）。

### 2. インストール

スクリプトをダウンロードし、実行権限を付与します。

```bash
chmod +x inspect_wp.sh
```

./inspect_wp.sh
